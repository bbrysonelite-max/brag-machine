# Business Promo Style (any business, any website)

A launch-quality promo for whatever business the VIDEO REQUEST names. The request is the brand brief — this file is the craft rules.

## Brand (adaptive)

| Element | Value |
|---|---|
| Background | default `#0A0E14` dark stage — shift the hue toward the business's brand color if the request gives one |
| Text | `#F5F7FA` primary · `#C7D2DE` secondary · `#8FA0B3` sub |
| Accent | THE BUSINESS'S color if given in the request; else warm gradient `#FFB36B → #FF6B1A` |
| Font | Inter 400/600/800/900 (bundled — never CDN) |
| Cards/UI | `#121A24` bg, `#22303E` border, radius 18–26px |
| Logo | recreate simple text-lockup of the business name; never fake a real logo mark |

## Creative Laws

1. **15–25 seconds.**
2. **Hook in the first 2–3s** — the customer's problem or a jolt of the outcome. Never open on the logo.
3. **Pattern:** hook → reveal (name + what it is in ONE line) → 2–3 sharp highlights → punchline + CTA end-card.
4. **Show the thing:** recreate a believable slice of the product/site UI on a card. No abstract filler.
5. **No generic SaaS language.** "Streamline your workflow" is banned. Use the request's real copy; if none given, write like a sharp founder, not a brochure.
6. **Sell the outcome, never the feature list.**
7. **Readable:** fast-in, then HOLD. Short label ≥0.8s settled; sentences ~0.3s/word.
8. **Beat-sync:** 1–3 major reveals on strong cues; sequential items snap to the grid.
9. **CTA end-card is mandatory:** the URL from the request (or --cta), big and alone for the final 1.5–2s.

## Voice

Match the business's audience, but keep Brent's spine: blunt, concrete, confident. Explain it so a civilian gets it.
