# Motivational Quote Style

Big-type quote films. One idea per video, said like Brent would say it — earned wisdom, not poster-hype.

## Brand

| Element | Value |
|---|---|
| Background | `#0A0E14` deep charcoal-navy, warm radial glow low in frame (`rgba(255,179,107,0.10)`) |
| Text | `#F5F7FA` primary · `#C7D2DE` secondary |
| Accent | warm ember gradient `#FFB36B → #FF6B1A` — used ONLY on the key word/phrase |
| Font | Inter — 900 for the quote, 400 for attribution. Quote type is HUGE: fills 70%+ of frame width |
| No logo, no product UI | this is a message, not an ad |

## Creative Laws

1. **15–20 seconds.** One quote, fully landed, beats three rushed.
2. **Hook = the sharpest fragment of the quote alone on screen** in the first 2s — then build the rest around it.
3. **Pattern:** fragment → full quote assembles (word-by-word or line-by-line on the beat) → HOLD 2s+ on the finished quote → quiet sign-off.
4. **One accent moment:** exactly one word/phrase gets the ember gradient + a scale or reveal beat. Everything else stays calm.
5. **No stock-poster clichés** ("grind", "hustle harder", generic mountain energy). The line must sound like a person who's actually done it.
6. **Readable above all:** big type, long holds. Fast-in, then HOLD. Never flashing.
7. **Beat-sync:** word/line reveals snap to the beat grid; the accent moment lands on a strong cue.
8. **Sign-off:** small, quiet — "— Brent Bryson" or the CTA if one is set. Never both loud.

## Voice

35+ years of business scars, said plainly. Truth first, comfort second. Short declarative lines. It should read like something a mentor said once and you never forgot.
